> 可以给当前获取短信验证码的按钮一个点击后的禁用标志 `class="disable" `

- **`a` 标签禁止跳转**
- **给 `body` 标签添加，防止在网页下拉过程中的误触事件发生**
- **给遮盖层添加 `pointer-event: none` 后可以操作被遮盖的元素**
