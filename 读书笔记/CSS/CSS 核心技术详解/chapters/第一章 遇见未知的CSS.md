### 3. 隐藏元素
- **通过 `width: 0` 和 `height: 0` 来隐藏元素**
> **元素本身的内容还存在**
> **缺点：** <mark>无法隐藏文字</mark>，需要更改元素的背景色以及文字的大小来配合
```css
div {
  width: 0;
  height: 0;
  background-color: transparent;
  font-size: 0;
}
```

- **将元素的 `opacity` 设置成 `0`**
> **元素本身还存在，只是看不见**

- **通过定位将元素移出屏幕范围**
> **元素本身还存在，只是移出屏幕范围**
```css
div {
  position: absolute;
  left: -9999px;
}
```
