## 一. 原始值与引用值
>变量两种不同类型的数据：**原始值**和**引用值**
>**原始值**（primitive value）:最简单的数据
>**引用值**（reference value）则是由多个值构成的对象
- 原始值是按值访问
- 引用值是按引用访问
- javascript中的字符串是按值访问的，与其他编程语言不同
- 原始值不能有属性

1. 复制值
```javascript
let obj1 = new Object();
let obj2 = obj1;
obj1.name = "Nicholas";
console.log(obj2.name); // "Nicholas"
```
